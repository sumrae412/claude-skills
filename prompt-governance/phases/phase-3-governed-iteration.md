# Phase 3: Governed Iteration

## Goal

Define the day-to-day workflow for changing prompts safely.

## Cover

- branch
- develop
- eval
- compare
- review
- promote
- monitor
- rollback

## Rules

- no prompt promotion without comparison to current production behavior
- monitor post-deploy, not just pre-deploy

## Output

Governed prompt-iteration workflow with roles, gates, and approval points.
