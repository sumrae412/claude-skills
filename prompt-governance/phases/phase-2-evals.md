# Phase 2: Evals

## Goal

Define how prompt quality is measured before changes reach production.

## Cover

- eval types
- golden datasets
- pass thresholds
- failure reporting
- CI integration

## Rules

- version the golden dataset with the prompt
- choose metrics that match the task shape
- call out where human eval is still required

## Output

Eval-pipeline design with dataset structure, scoring method, and release
gates.
