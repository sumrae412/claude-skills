# Phase 3: Comparison Analysis

## Goal

Compare working vs. broken states when a viable baseline exists.

## Compare

- call order
- initialization
- configuration
- data shapes
- error handling
- dependency versions

## Rules

- if no working reference exists, say so explicitly
- record every meaningful delta even if it may later prove irrelevant

## Output

Structured working-vs-broken comparison table or explicit note that no
baseline exists.
