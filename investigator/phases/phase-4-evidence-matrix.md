# Phase 4: Evidence Matrix

## Goal

Assemble findings into the final evidence matrix.

## Include

- problem type
- affected area
- factual observations with confidence
- comparison analysis, if any
- sources checked with no findings
- open questions

## Rules

- confidence reflects evidence strength, not conviction
- stop before diagnosis, prioritization, or solutioning

## Output

Final evidence matrix ready for a caller like `bug-fix`.
