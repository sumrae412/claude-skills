# Stage 2: Product Scan

Load `../references/review-prompts.md` before running this stage.
Load `../references/ux-persona-schema.md` only for UX/usability personas
that will drive website or Playwright-style walkthroughs.

## Goal

Build the product/context map that the persona will react to.

## Ask

- Should the current codebase be scanned, or are review materials
  provided directly?
- Is a journey map needed, or only a product map?

## Build

- scan user-facing materials
- organize findings by product areas, not code architecture
- optionally build a journey map
- enrich the persona lens based on the actual product

## Output

Product map plus persona-specific focus areas.
