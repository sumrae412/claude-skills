# Stage 3: Interactive Session

Load `../references/review-prompts.md` before running this stage.

## Goal

Run the persona in one of four interaction modes.

## Modes

- Product Review
- Brainstorming
- Rehearsal
- Dynamics Analysis

## Ask

- Which mode is needed?
- Which saved persona should be used?
- Does the user want strict in-character behavior, or a lighter advisory
  stance?

For dynamics analysis, require the user's own profile or equivalent
grounding first.

## Output

In-character session plus any low-confidence flags or debrief notes.
