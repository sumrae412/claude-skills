# Multi-Persona Features

Load this only when 2+ saved personas already exist.

## Goal

Compare personas or evaluate one feature/decision across several personas.

## Modes

- comparison matrix
- feature decision framework

## Use When

- choosing which persona to deploy for review
- finding cross-persona agreement or disagreement
- checking whether one feature helps one persona and harms another

## Output

Either:

- side-by-side persona matrix, or
- per-persona feature verdict with overall recommendation
