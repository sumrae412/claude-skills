# Length Variations + Context Adaptations

Load when the user wants a specific length or a non-investor context (networking,
cold outreach, meeting opener, follow-up). Every version keeps the Villain-Hero
shape; shorter versions compress, they do not drop the villain.

## 10-second one-liner

Pick one shape:

- **Action:** "We're [Company] — we [verb] for [audience] so they can [outcome]."
- **Analogy:** "We're the [known company] of [your market]."
- **Problem-solution:** "[Big number] [people/companies] waste [resource] on
  [problem]. We [solution]."

The analogy shape is the riskiest for sameness — only use it when the comparison
is exact and the listener knows the reference company.

## 30-second pitch

```
[0:00-0:10] VILLAIN   "[Shocking stat or one-line story about the problem]"
[0:10-0:20] HERO      "[Company] [solves how]. [The one differentiator]."
[0:20-0:30] VICTORY + ASK
                      "[Traction proof]. We're raising [amount] for [milestone]."
```

## 60-second pitch

Use the matching investor-type script in
[`investor-types.md`](investor-types.md). Each is timed to 60 seconds with
villain / hero / victory / ask markers.

## 90-second pitch

Take the 60-second investor script, then add one expanded-proof beat before the
final ask:

```
[after the ask, +0:30] EXPANDED PROOF & VISION
"One example: [specific customer story with numbers]. What excites us most is
[vision]. As [market trend] accelerates, we're positioned to [strategic
opportunity]. Our next milestone is [specific goal]; with this raise we'll
[concrete achievement]. Can we put 30 minutes on the calendar to go deeper?"
```

## Context adaptations

| Context | Lead with | Body | Close with |
|---|---|---|---|
| **Conference / networking** | A curiosity hook, conversational not performative | Compressed 30s | An exchange — card, LinkedIn, a specific next touch |
| **Cold email / LinkedIn** | Subject line = villain hook; first line = relevance to *this* recipient | Compressed 30s | One low-friction, specific ask |
| **Investor meeting opener** | Acknowledge their portfolio / thesis | Villain framed as "you've probably seen this" | Sets up the deck walkthrough |
| **Follow-up / second meeting** | Reference the prior conversation | Address the concern they raised + new proof | Advance the narrative, name the decision |

For cold outreach, the relevance line is load-bearing: a generic opener gets
deleted. Tie the villain to something specific about the recipient.
