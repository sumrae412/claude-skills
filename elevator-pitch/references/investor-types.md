# Investor-Customized 60-Second Pitches

Load when the audience is a specific investor type. Each script is timed to 60
seconds and emphasizes what that investor actually optimizes for. Same
Villain-Hero spine; different proof currency.

Pick the version that matches the investor, do not blend all three — a blended
pitch reads as unfocused.

## A. Technical investor

For technical VCs, engineers-turned-investors, deep-tech funds.

| Element | Focus |
|---|---|
| Villain | Technical complexity / legacy stack |
| Hero | Innovation, IP, architecture |
| Proof | Technical moat (speed, accuracy, defensibility) |
| Metrics | Performance numbers, patents, benchmarks |

```
[0:00-0:15] VILLAIN
"[Industry] runs on [outdated approach]. The result is [technical pain with
specific numbers]. It's not that no one tried — [prior attempts] failed because
[technical reason]."

[0:15-0:40] HERO
"We built [Product]. Unlike [alternatives], we [technical differentiator]. Our
team includes [technical credibility]. We've already [technical achievement]."

[0:40-0:55] VICTORY
"[Customer / pilot] is seeing [specific technical improvement]. Our architecture
unlocks [future capability]."

[0:55-0:60] ASK
"We're raising [amount] to [technical milestone]. I'd love to walk you through
[technical deep-dive offer]."
```

## B. Market-focused investor

For growth VCs, business-focused angels, institutional investors.

| Element | Focus |
|---|---|
| Villain | Market inefficiency |
| Hero | Business model |
| Proof | Traction, growth rate |
| Metrics | Revenue, CAC/LTV, TAM |

```
[0:00-0:15] VILLAIN
"[Industry] is a $[X]B market where [painful inefficiency]. Companies waste
$[Y] a year on [workaround]. No one has solved it because [barrier / insight]."

[0:15-0:40] HERO
"[Company] is [one-line description]. We found that [insight]. Our model gives
us [business-model advantage]. We're seeing [key metric]."

[0:40-0:55] VICTORY
"We launched [timeframe] ago with [traction proof]. Customers include [notable
names / segments]. We're growing [growth rate]."

[0:55-0:60] ASK
"We're raising [amount] to [growth milestone]. [Urgency, if real — round
closing, competitive timing]."
```

## C. Customer-obsessed investor

For consumer investors, impact funds, customer-centric VCs.

| Element | Focus |
|---|---|
| Villain | User pain, told as a story |
| Hero | The transformation |
| Proof | User love |
| Metrics | Retention, NPS, testimonials |

```
[0:00-0:15] VILLAIN
"Meet [persona]. Every [time period], [they] fight [problem]. [Emotional
detail]. [They're] not alone — [X million] people hit this exact wall."

[0:15-0:40] HERO
"Now [persona] uses [Product]. Instead of [old way], [they] just [new way].
[Quote or reaction]. What changed? We [insight / innovation]."

[0:40-0:55] VICTORY
"[Persona] isn't unusual. Our [number] users [engagement proof].
[Testimonial snippet]. NPS is [score], with [retention metric]."

[0:55-0:60] ASK
"We're raising [amount] to bring this to [expansion]. [User-centric vision]."
```

## Choosing the version

- They ask about the tech first / have an engineering background → A.
- They ask about the market, margins, or growth → B.
- They ask "who's your user" / fund consumer or impact → C.

If you do not know the investor's lean, default to B and keep one technical proof
point and one user quote in reserve for the Q&A.
