# Phase 3: HIPAA and Privacy

Load `../references/hipaa_compliance_framework.md` before running this
phase.

## Goal

Determine whether HIPAA obligations apply and what safeguards are needed.

## Ask

- Does the device create, store, transmit, or expose ePHI?
- Is there EHR integration or cloud transmission?
- Which administrative, physical, and technical safeguards already exist?
- Are BAAs or covered-entity relationships involved?

Use `scripts/hipaa_risk_assessment.py` when structured evaluation helps.

## Output

HIPAA applicability and safeguards assessment with high-risk privacy and
security gaps.
