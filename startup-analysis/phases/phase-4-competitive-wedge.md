# Phase 4: Competitive Wedge

## Goal

Map competitors and define the most credible unclaimed positioning axis.

## Cover

- competitor table
- current behavior / status quo as the real enemy (habit, spreadsheet, internal process, agency, existing tool)
- 3 unclaimed axes
- emotional benefit
- target audience for each axis
- proof strategy
- wedge one-liner

## Rules

- Treat "we have no competition" as false by default. If no direct competitor exists, the real enemy is current behavior.
- Differentiation must be specific. "Better" and "cheaper" do not count.

## Output

Competitive wedge analysis with recommended lead axis and strategist's
note.
