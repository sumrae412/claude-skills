# Resume Bullet Bans

Load this file during Phase 2 matching and before final export. These patterns down-level the resume even when the underlying work is strong.

## 1. Banned Openers

Rewrite bullets that begin with:

- Responsible for
- Helped with
- Assisted with
- Worked on
- Involved in
- Tasked with
- Participated in
- Supported

These describe proximity, not contribution.

## 2. Weak Verbs and Filler

Rewrite or remove:

- leveraged
- utilized
- drove strategy
- handled
- managed end-to-end
- collaborated with cross-functional teams
- owned various initiatives

These can be valid ideas, but they are too vague unless the bullet immediately names what changed because of them.

## 3. Tool Piles

Reject bullets that are mostly a list of tools, stacks, or buzzwords without an outcome.

Weak:

- Built LLM, RAG, agentic, Python, AWS solutions for enterprise use cases.

Better:

- Built a retrieval-backed workflow in Python and AWS that reduced analyst turnaround time and gave product teams a reusable pattern for later launches.

## 4. Hidden Scope Inflation

Watch for phrases like:

- led strategy
- drove transformation
- delivered innovation
- enabled stakeholders

If the bullet uses one of these, it must also name:

1. what concrete mechanism was introduced
2. what problem or tradeoff it addressed
3. what changed after

If not, the bullet is empty.

## 5. Final Resume Bullet Test

For each bullet, ask:

1. Is the subject doing something concrete?
2. Is there a visible result, constraint, or decision?
3. Could a hiring manager picture what changed?

If not, rewrite or cut it.
