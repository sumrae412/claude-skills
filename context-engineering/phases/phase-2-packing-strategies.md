# Phase 2: Packing Strategies

## Goal

Package context in a way the agent can use without drowning in it.

## Strategies

- brain dump
- selective include
- hierarchical summary

## Rules

- include only what the current task needs
- provide one pattern example when style matters
- keep task-specific packs explicit about files, constraints, and examples

## Output

Recommended packing pattern and a draft structure for the current session.
