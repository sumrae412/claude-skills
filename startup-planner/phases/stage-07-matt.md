# Stage 7: MATT

Load `../references/kawasaki-frameworks.md` before running this stage.

## Goal

Create a tactical framework for milestones, assumptions, tests, and
tasks.

## Milestones

Use the core five:

1. Working prototype
2. Initial capital
3. Field-testable version
4. Paying customer
5. Cash-flow breakeven

## Ask

- Which milestone matters next?
- Which assumption, if wrong, kills the business?
- How will you test demand, pricing, and acquisition?
- What must happen this week and this month?

## Push

- Prefer a short list of killing assumptions over a long risk dump.
- Turn vague tests into observable pass/fail checks.

## Output

MATT document with current status, key assumptions, planned tests, and
immediate tasks.
