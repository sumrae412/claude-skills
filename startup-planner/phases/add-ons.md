# Optional Add-Ons

Load only when the current stage benefits from the extra structure.

## PRO / CON Debate Validation

Best at:

- Stage 2: Sweet Spot
- Stage 9: Market Analysis

Load `../references/debate-technique.md`.

Use when:

- The opportunity feels emotionally compelling but weakly evidenced.
- There is a strong incumbent or substitute.
- The user needs a pressure test before committing to scope.

## Customer Segmentation Matrix

Best at:

- Stage 9: Market Analysis

Load `../references/segmentation-matrix.md`.

Use when:

- The market has multiple user behaviors.
- One persona is clearly too simplistic.
- Feature priorities depend on segment mix.

## AI Stakeholder Clone and Rehearsal

Best at:

- Stage 12: Pitch Deck

Load `../references/stakeholder-clone.md`.

Use when:

- The user is pitching a known stakeholder.
- Objection handling matters as much as slide quality.
- The user needs a rehearsal pass before presenting.
