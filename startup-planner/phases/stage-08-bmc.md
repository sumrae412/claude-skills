# Stage 8: Business Model Canvas and Value Proposition

Load `../references/business-frameworks.md` before running this stage.

## Goal

Structure the business model using a Value Proposition Canvas and a
one-page Business Model Canvas.

## Part A: Value Proposition Canvas

Build the customer profile:

- Customer jobs
- Pains
- Gains

Build the value map:

- Products and services
- Pain relievers
- Gain creators

Use the value proposition statement template from the reference.

## Part B: Business Model Canvas

Walk through all 9 components:

1. Customer Segments
2. Value Propositions
3. Channels
4. Customer Relationships
5. Revenue Streams
6. Key Resources
7. Key Activities
8. Key Partnerships
9. Cost Structure

## Push

- Keep each BMC box to one sentence where possible.
- Keep the full canvas on one page.
- If the value proposition is muddy, pause before filling the BMC.

## Output

Completed Value Proposition Canvas plus a one-page Business Model
Canvas.
