# Stage 12: Pitch Deck

Load `../references/business-frameworks.md` before running this stage.

## Goal

Create an investor-ready Marp markdown pitch deck from earlier outputs.

## Deck Shape

Build 12-15 slides covering:

1. Title
2. Problem
3. Solution
4. Market Opportunity
5. Product / MVP
6. Business Model
7. Competitive Advantage
8. Go-to-Market Strategy
9. Financial Projections
10. Team
11. Milestones and Roadmap
12. Ask / Next Steps

## Rules

- One key message per slide
- 3-5 bullets max
- About 500 characters max per slide
- Cite data sources

Use the Marp guidance from `business-frameworks.md`.

## Push

- Strip filler.
- Keep slides evidence-backed.
- Make sure the ask is explicit.

## Output

Complete Marp markdown deck ready for export.

## Optional Add-On

Load `add-ons.md` if a stakeholder rehearsal would sharpen the pitch.
